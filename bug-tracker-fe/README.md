# Bug Tracker Front-end

This is a public, unauthenticated bug report form built with Next.js and Tailwind CSS.

## Getting Started

1.  **Install dependencies:**
    ```bash
    npm install
    ```

2.  **Set up environment variables:**
    Create a `.env.local` file (one has been provided with a default).
    ```env
    NEXT_PUBLIC_API_URL=http://localhost:3000
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

4.  **Open the app:**
    Navigate to [http://localhost:3001](http://localhost:3001) (or the port specified by Next.js).

## Features

- **Public Access:** No authentication required to submit bug reports.
- **Dynamic Feedback:** Success, validation, and network error messages.
- **Responsive Design:** Styled with Tailwind CSS for mobile and desktop.
- **Priority Support:** Choose between High, Medium, and Normal priorities.
